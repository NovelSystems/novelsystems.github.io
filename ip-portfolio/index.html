Patent Portfolio

IP portfolio summary

Novel Systems Engineering is building a patent portfolio covering the core DDR measurement method, signal processing pipeline, sensor hardware architecture, and application-specific implementations. The portfolio is structured to support licensing across all three target markets.

The IP strategy prioritizes broad method claims on the DDR measurement principle, supported by narrower implementation claims on the 1D silicon photonic phased array architecture and the FMCW signal processing pipeline. Application-specific claims cover the maritime PIW, drone detection, and radar augmentation implementations separately, allowing licensing to be structured per market without cross-licensing entanglements.

All applications listed below are in active prosecution or preparation. Patent numbers and publication dates will be added as applications publish. Licensing inquiries may reference applications by their subject-matter title pending publication.

Polarimetric LiDAR Target Classification Using Differential Depolarization Response

Core method claim covering the DDR measurement principle: simultaneous acquisition of orthogonal polarization returns from a shared aperture and computation of the depolarization ratio asymmetry for target classification. Covers all wavelengths, all platform types, and all target classes.

Application — In Preparation

FMCW Signal Processing Pipeline for DDR Computation with Range-Gated Clutter Rejection

Implementation claim covering the FMCW modulation and coherent detection architecture that enables simultaneous range gating and DDR computation. Claims the combination of FMCW ranging with polarimetric dual-channel coherent detection for real-time classification.

Application — In Preparation

1D Silicon Photonic Phased Array Pair Architecture for Polarimetric LiDAR

Hardware claim covering the use of two orthogonally-oriented 1D phased arrays as a polarimetric aperture pair, the post-test pairing methodology for yield improvement, and the mechanical second-axis steering scheme that enables 2D scan from two 1D dies.

Application — In Preparation

Automated Person-in-Water Detection System Using Polarimetric LiDAR Classification

Application-specific claim for the maritime PIW use case. Covers the system architecture, classification pipeline, and operator interface for shipboard and airborne PIW detection using DDR measurement. Includes claims on the alert and position-output protocol.

Application — In Preparation

Radar Track Augmentation via Polarimetric LiDAR Material Classification Overlay

Application-specific claim for the radar augmentation use case. Covers the synchronized pulsed ToF LiDAR system, the DDR classification overlay, and the track-layer fusion protocol for adding material classification attributes to conventional radar contacts.

Application — In Preparation

UAS Classification Using Differential Depolarization Response Against Low-Clutter Airspace Background

Application-specific claim for the drone detection use case. Covers the detection and classification method for composite-airframe UAS against atmospheric background, the clutter discrimination approach, and integration into C-UAS sensor fusion architectures.

Application — In Preparation
