Licensing Domains

Applications

DDR polarimetric LiDAR has three licensing domains: maritime person-in-water detection, drone and bird discrimination, and radar augmentation. Each domain is grounded in a specific background medium that produces near-zero DDR, providing a stable physical reference against which target contrast is measured.

Maritime

FMCW Maritime LiDAR

Ocean water is a volume scatterer. Under orthogonal H and V illumination, it produces symmetric depolarization, giving DDR near zero. Manufactured materials and biological structures floating on the surface produce nonzero DDR. The ocean surface functions as the calibration reference, not an interference source.

This property makes DDR-enabled LiDAR a candidate for person-in-water (PIW) detection in maritime search and rescue. The system classifies targets by DDR contrast rather than image interpretation. No trained operator is needed to read sensor returns; the output is a binary detection flag with position data for handoff to rescue crews. The 1550 nm wavelength is eye-safe at all operational ranges.

The system is designed for integration into shipboard sensor mast installations and helicopter-mounted sensor pods. Licensing is available for both platform types under separate royalty structures.

Drone Detection

FMCW Drone Detection

Clear atmosphere at 1550 nm produces near-zero DDR background. The absence of a surface scatter component makes this background lower and more stable than the maritime case. Carbon fiber and fiberglass drone airframes produce strong DDR signatures from their anisotropic layered structure. Bird feathers produce a different DDR signature from biological keratin microstructure. Phase I will characterize the separation between these signatures experimentally; the discrimination is predicted by physics but not yet measured.

DDR adds a material classification channel to airspace surveillance. The stable atmospheric background allows classification thresholds to be set more sensitively than in the maritime case. Output is a classification flag and track initiation data for integration with existing counter-UAS sensor fusion systems.

Radar Augmentation

Pulsed ToF Radar Augmentation

Radar measures geometry and motion. It cannot measure material composition. DDR measures material composition directly through polarimetric asymmetry. These are complementary channels: radar provides track position and kinematics; DDR adds material classification to each contact.

The integration model is a secondary classification layer cued by radar contact position. A pulsed time-of-flight LiDAR aperture operates in the same pointing direction as the host radar and is synchronized to its scan pattern. Classification output is fused into the radar track management layer through a standard RS-422 or Ethernet data interface. No radar hardware modification is required.

Target platforms include coast guard stations, offshore facilities, port installations, and naval vessels.

  • Ranging: pulsed ToF, synchronized to host radar
  • Classification: material DDR overlay per radar contact (design target)
  • Integration: RS-422 or Ethernet, track-layer fusion

  • Target platforms: coast guard, offshore, port, naval
  • Range: matched to host radar detection envelope (design target)
